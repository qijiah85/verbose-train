# 正式技能目录

将可复用、可分发的 Agent 技能放在本目录。每个子目录对应一个技能。

```text
skills/
└── <skill-name>/
    ├── SKILL.md
    └── <optional-supporting-files>
```

新建时复制 [`../templates/SKILL.md`](../templates/SKILL.md)，并更新上级 [`../README.md`](../README.md) 索引。
