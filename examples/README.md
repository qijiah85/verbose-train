# 示例（Examples）

本目录提供**可复制改造**的完整示例，帮助贡献者快速上手。

示例仅作演示，不作为现行生产规范或正式技能；正式内容请放入：

- 规范 → [`../specifications/`](../specifications/)
- 技能 → [`../agent-skills/skills/`](../agent-skills/skills/)

## 内容

| 路径 | 说明 |
|------|------|
| [`specifications/sample-coding-standard.md`](./specifications/sample-coding-standard.md) | 规范文档完整示例 |
| [`agent-skills/hello-world/`](./agent-skills/hello-world/) | 最小可用 Agent 技能示例 |

## 使用方式

1. 阅读示例，理解结构与必填字段
2. 复制到正式目录并改名
3. 替换为真实内容，将 `status` 从示例说明改为 `draft` / `active`
