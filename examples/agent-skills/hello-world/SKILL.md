# Hello World 技能示例

> 本技能位于 `examples/agent-skills/`，仅作演示。正式技能请放到 `agent-skills/skills/`。

## 概述

展示一个合规技能包的最小形态：目录名、`SKILL.md` 入口、frontmatter 与分节正文。

## 适用场景

**使用本技能，当：**

- 你想确认本仓库的技能文件布局是否正确
- 你需要一个可复制的最小模板对照

**不要使用本技能，当：**

- 需要解决真实工程问题（请改用或编写正式技能）

## 流程

1. **确认目录**：技能位于独立目录，入口为 `SKILL.md`
2. **确认元数据**：`name` 与目录名一致；`description` 含触发场景
3. **确认正文**：包含概述、场景、步骤与红线
4. **输出问候**：若用户明确要求演示，回复「Hello, Agent Skill!」并简述结构要点

## 红线

- 不要把本示例当作生产技能加载到关键工作流
- 不要省略 `description` 或使用模糊描述（如「通用帮助」）

## 示例

### 输入

> 请用 hello-world 技能演示一下技能结构。

### 期望行为

Agent 按上述流程检查结构，并返回简短演示回复。
